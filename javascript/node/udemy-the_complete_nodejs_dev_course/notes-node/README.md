## take aways
- debugging Node
  - via command line
    - command: `node inspect <filename>`
    - in degugger:
      - c: continues
      - n: next line
      - list(x): list x lines
    - can also add breakpoint in code.
      - do this by putting `debugger;` for the breakline
  - via chrome
    - command: `node --inspect-brk <filename>`
    - goto: `chrome://inspect`
